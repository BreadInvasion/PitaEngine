The main function. 

\begin{DoxyAuthor}{Author}
Mark Haddleton
\end{DoxyAuthor}
This is the main function of the entire engine. It contains function calls to start\+: In-\/game time, the heap, the file system, the game window, and the renderer. Once those are in place, the game is initialized, then updated every frame in the core \char`\"{}pump\char`\"{} loop. Upon shutdown, all of the allocated memory blocks are freed and the window is closed.
