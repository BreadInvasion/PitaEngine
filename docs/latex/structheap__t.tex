\doxysubsection{Detailed Description}
A structure to define the engine\textquotesingle{}s heap. 

The struct \mbox{\hyperlink{structheap__t}{heap\+\_\+t}} is used to represent the heap, the space within the engine that manages all memory allocation/de-\/allocation over the course of the program\textquotesingle{}s execution. This implementation of heap uses a tlsf implementation written by Matthew Conte to handle all memory operations.
