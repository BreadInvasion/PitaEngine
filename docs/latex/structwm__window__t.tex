\doxysubsection{Detailed Description}
\mbox{\hyperlink{structwm__window__t}{wm\+\_\+window\+\_\+t}} is a structure to store data for the OS-\/level window. 

It contains important info, including the raw window object (HWND), a back reference to the heap, and other states like the mouse position, quit and focus states, and input masks.
