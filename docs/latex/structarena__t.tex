\doxysubsection{Detailed Description}
A structure to represent \char`\"{}arenas\char`\"{} within a heap. 

The struct \mbox{\hyperlink{structarena__t}{arena\+\_\+t}} is used to represent specific \char`\"{}arenas\char`\"{}, pools of contiguous memory within the heap that can be divided up into blocks, to then be allocated and destroyed as necessary. Arenas are commonly split up based on tasks within the game, so that a task\textquotesingle{}s memory can be freed all at once when that task is complete, but in this specific implementation we just assign new heaps whenever we run out of memory in the current heap.
