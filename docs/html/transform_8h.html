Detailed Description

Author
Mark Haddleton

Transforms manage the position, orientation, and relative size of objects in 3D space - this file contains a structure and associated methods to handle and alter these values both independently and in relation to eachother.

Typedef Documentation

◆ transform_t

typedef struct transform_t transform_t

transform_t stores spatial positioning data.

Specifically, linear position, scale, and rotation are stored as vectors and a quaternion.

◆ transform_invert()

void transform_invert ( transform_t *  transform)

Inverts the provided transform.

Scale is simply a multiplicative inverse, Rotation uses the function quatf_conjugate(), and the translation is the additive inverse, adjusted for rotation, then multiplied by the new scale.

Parameters
transformThe transform to be inverted, which will be reassigned the new values.

◆ transform_multiply()

void transform_multiply ( transform_t *  result,
const transform_t *  t 
)

Combines two input transforms.

This is not as simple as adding the two together - while the scales and rotations can simply use vector and quaternion multiplication respectively, the translation is a little bit more complicated. First, one of the translations has to be multiplied by the other's scale and rotation, reflecting the difference in relative positioning. Once they are in the same relative scope, then they can be added together.

Parameters
resultThe first transform provided, which will also be replaced with the output.
tThe second transform provided.
