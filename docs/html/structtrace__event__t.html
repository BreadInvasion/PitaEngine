Detailed Description

trace_event_t stores the process info related to a specific trace event.

trace_event_t also contains a pointer next, resulting in a self-contained singly linked list.

Field Documentation

◆ name

const char* name

name is equal to the name of the event.

◆ next

trace_event_t* next

next is a pointer to the next chronological event to occur, if one exists - otherwise, null.

◆ ph

char ph

ph is 'B' if marking the start event, or 'E' if marking the end event.

◆ pid

DWORD pid

pid is equal to the Process ID at time of event.

◆ tid

tid is equal to the Thread ID at time of event.

◆ ts
