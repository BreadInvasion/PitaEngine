Detailed Description

wm_window_t is a structure to store data for the OS-level window.

It contains important info, including the raw window object (HWND), a back reference to the heap, and other states like the mouse position, quit and focus states, and input masks.

Field Documentation

◆ has_focus

bool has_focus

has_focus changes state when the manager receives the WM_ACTIVATEAPP flag, becoming true if the window is currently in focus. Otherwise, has_focus is false. This flag is used to enable or disable mouse position updates.

◆ heap

heap_t* heap

heap is a reference to the heap in which the window object is stored.

◆ hwnd

HWND hwnd

hwnd stores the raw OS-level window object.

◆ key_mask

uint32_t key_mask

key_mask is a bit mask that records up/down states for the up, down, left, and right keyboard keys.

◆ mouse_mask

uint32_t mouse_mask

mouse_mask is a bit mask that records up/down states for the left, right, and middle mouse buttons.

◆ mouse_x

int mouse_x

mouse_x records the relative x position of the mouse within the window, updating every frame as long as the window is currently in focus.

◆ mouse_y

int mouse_y

mouse_y records the relative y position of the mouse within the window, updating every frame as long as the window is currently in focus.

◆ quit

bool quit

quit becomes true if the manager receives a WM_CLOSE flag. Otherwise, quit is false.
